# 🎮 Tic Tac Toe Game

A modern and responsive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. Play against a friend, enjoy smooth gameplay, and experience a clean and user-friendly interface.

---

## 🚀 Features

* 🎯 Two-Player Gameplay
* 🏆 Automatic Winner Detection
* 🤝 Draw Detection System
* 🔄 New Game Functionality
* ♻️ Reset Game Option
* 📱 Responsive Design
* ✨ Clean and Attractive User Interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

---

## 🎮 How to Play

1. Player **O** starts the game.
2. Players take turns marking spaces in the 3×3 grid.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled and no player wins, the game ends in a draw.
5. Use **New Game** or **Reset Game** to start again.

2. Navigate to the project folder:

```bash

```

3. Open `index.html` in your browser.
